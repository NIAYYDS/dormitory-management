# 智慧宿舍管理系统

基于浏览器的智慧宿舍管理系统，支持管理员和住宿人员两种角色。

## 功能特性

### 管理员功能
- 📊 **数据总览**：宿舍总数、床位使用率、入住率、待处理报修统计
- 🏠 **宿舍管理**：添加/删除宿舍、入住登记/退宿、床位分配
- 🔧 **报修管理**：待处理/维修中/已完成状态管理、上传维修照片

### 住宿人员功能
- 🏠 **我的宿舍**：查看宿舍信息、床位信息、室友信息
- 🔧 **报修服务**：提交报修、上传损坏照片、查看维修进度

## 技术特点

- 🌐 **纯前端实现**：无需后端服务器，数据存储在浏览器 localStorage
- 📱 **响应式设计**：完美适配桌面端和移动端
- 🎨 **精美UI**：温暖的配色方案，流畅的动画效果
- 📷 **照片上传**：支持拍照上传，图片自动压缩
- 📱 **二维码分享**：内置二维码生成，手机扫码即可访问

## 使用方法

### 快速本地使用
直接用浏览器打开 `index.html` 文件即可使用。

### 默认登录
- **管理员密码**：`admin123`
- **住宿人员**：选择宿舍 → 选择姓名 → 登录
